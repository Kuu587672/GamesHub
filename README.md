# GamesHub

GamesHub is a personal web project focused on building fun, interactive browser games using **pure HTML, Tailwind CSS, and JavaScript**.  
The goal of this project is to create a growing hub of simple yet engaging games while improving frontend development skills.

**Live Demo:** https://Kuu587672.github.io/GamesHub/

---

## Features

- Modern dark gaming UI
- Fully responsive design
- Smooth scrolling navigation
- Mini games built with vanilla JavaScript
- Hosted using GitHub Pages

---

## Games Available

### 1. Number Guessing Game
- Guess a random number between **1 and 100**
- Get instant feedback: *Too High / Too Low*
- Attempt counter
- Reset option to play again

More games will be added soon
